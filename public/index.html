nimble-explorer

Search for Nimble packages



Welcome

This is a tool to search for libraries for the language Nim which can be installed with its repository manager Nimble. To get started, fill out the form above. Report any issues on the Github page.

Tips

You can search for particular attributes with the colon operator. e.g tags:python in order to find entries which have been tagged with Python. Note that the colon operator is space sensitive. You can use the following for the left hand of the colon operator: names, tags, description and license.

Unfortunately your browser does not appear to support technologies which this site relies on. While this will be resolved in the future, current official support is for Firefox and Chrome - in particular, Safari under iOS does not appear to work.
